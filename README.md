fastpkg(1) -- Simple suckless package manager
=============================================

## SYNOPSIS

`fastpkg` [<optional>...] <flags>

`fastpkg-template` [-h, --help] <FILE_NAME> 

## DESCRIPTION

Add packages directly from their original URL.

## OPTIONS

* `-h`, `--help` :
  Displays the help screen.

## EXAMPLES

List all packages

    $ fastpkg -a list
