fastpkg(1) -- A complementary package manager
=============================================

## SYNOPSIS

`fastpkg` [<optional>...] <flags>

`fastpkg-template` [-h, --help] <FILE_NAME> 

## DESCRIPTION

A complementary package manager, turn any URL-file into a package

## OPTIONS

* `-h`, `--help` :
  Displays the help screen.

## EXAMPLES

List all packages

    $ fastpkg -a list
